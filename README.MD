# Edubest Boutiques

A single-page luxury boutique website — quiet-luxury aesthetic, fully responsive, zero dependencies.

## Running it

Open `index.html` directly in any browser. There is no build step and nothing to install.

## What's inside

`index.html` is fully self-contained (HTML + inlined CSS + a little vanilla JS):

- Fixed nav that turns frosted-glass on scroll, with a mobile slide-in menu
- Hero, scrolling marquee, collections grid, brand story, services, testimonial
- Booking form with floating labels, live validation, and a personalized confirmation
- Scroll-reveal animations and an auto-updating copyright year

## Make it yours

Search `index.html` for these placeholders and swap in the real details
(there's a `TODO` comment in the footer block too):

- **Contact** — address, hours, phone, and `concierge@edubestboutiques.com`
- **Founder line** in the story section
- **Collection names** (`L'Heure d'Or`, `Maison Noir`, `Jardin Privé`)
- **Collection images** — the cards use CSS gradient placeholders; drop in real
  photos by replacing the `.ph-1/2/3` backgrounds with `background-image: url(...)`
